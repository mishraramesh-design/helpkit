# HelpKit — Plug-and-play AI Assistant + L1/L2/L3 Support Widget

## What this is
Standalone product. Admin feeds it docs/code → AI builds knowledge bases → generates embed snippets for any app.

## Two services
- **assistant** (port 8092): Page-aware AI guidance widget
- **support** (port 8090): L1/L2/L3 error-code support chat widget
- **admin UI** (port 8091): Single admin panel managing both services
